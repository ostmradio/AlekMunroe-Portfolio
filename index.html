The Story of Kate Webster

As part of a second-year university project, I had the opportunity to work on The Story of Kate Webster, a short narrative-driven game built in Unity. You play as Kate Webster, making choices through a branching dialogue system that influences the story's outcome. I was the lead programmer in a group of three, responsible for coding the entire game. It was challenging deciding what features to prioritize in a short development window, but I’m really proud of what I achieved.

You can watch development videos here or download the project here.

Unity Dialogue System

While developing The Story of Kate Webster, I created a fully functional branching dialogue system from scratch using Unity’s xNode framework. Even though it started as part of a group project, I independently programmed and debugged the entire system. It took a lot of time and effort to add all the features and squash bugs, but I’m really happy with how it turned out. I plan to reuse and expand this system for future projects.

Wiinite

Wiinite started as a fun project after I was inspired by a video imagining what Fortnite would look like if it was made on the Nintendo Wii. I loved the concept so much that I decided to build my own version! The journey wasn’t easy — importing the Wuhu Island map, fixing texture issues, adding characters, and setting up emotes all had their challenges. But after a lot of trial, error, and creativity, I brought Wiinite to life. This project taught me so much about problem-solving and game development. If you want to see more about the project and its development journey, I made two videos on my YouTube channel explaining everything!

The Dark Lord's Reign

Structurefy

Structurefy is a simple but useful tool I built during my second year at university. It's a Unity editor extension that automatically generates a folder structure at the start of any new Unity project, based on a config file. No more creating folders by hand! While it wasn’t the most complicated thing I've made, it definitely saves time.
